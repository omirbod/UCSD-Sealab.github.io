Getting Started: Software

The software subsystem deals with all aspects of the control of the Stingray, including vision, navigation, path planning, pose estimation.

Robot Operating System (ROS)

The basis for all of the Stingray's software is the Robot Operating System (ROS). ROS is a platform that is widely used in the robotics and research community. ROS provides drivers for many off-the-shelf components and sensors, package management systems, visualization software, libraries for path planning, computer vision, and more, and many other features that are essential for robot software.

1. Install Ubuntu

ROS runs on Ubuntu Linux. To work on the software, you'll need to install Linux on your computer. A good way to do this is to use a virtual machine. VirtualBox is free software that will allow you to create virtual machines on Windows or Mac OS X. Other (not free) options are Parallels (for Mac) and VMware Fusion.

2. Install ROS

Once you have a working copy of Ubuntu, you can install ROS. The most current version of ROS is ROS hydro, but currently the Stingray's software is in ROS Groovy [are we switching? decide on this -- which to install!].

3. Go through ROS Tutorials

ROS supports C++, Python, and Lisp. We develop our software primarily in C++ and Python. Once you have ROS installed, work through the ROS tutorials in your language of choice to get familiar with ROS.

Github

Github is a website interface for the powerful version control system Git. Git manages your source code by storing a snapshot of your code base every time you commit to the remote repository. These snapshots keep track of all the revisions made to the software in the development cycle. Git also allows many developers to work on the same code, without having to be in the same place.

4. Sign up for Github

Create a Github account and username at Github.com. Email your username to Perry or Antonella so they can add you to the UCSD E4E Organization so you can start working on the software.

5. Work through the Github tutorials

Get familiar with Git and Github by reading about Git Basics and working through the Github tutorials.
