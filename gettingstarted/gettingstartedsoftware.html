Robot Operating System (ROS)

The basis for all of the Stingray's software is the Robot Operating System (ROS). ROS is a platform that is widely used in the robotics and research community. [short explanation "middleware"]

1. Install Ubuntu

ROS runs on Ubuntu Linux. To work on the software, you'll need to install Linux on your computer. A good way to do this is to use a virtual machine. VirtualBox is free software that will allow you to create virtual machines on Windows or Mac OS X. Other (not free) options are Parallels and VMware Fusion.

2. Install ROS

Once you have a working copy of Ubuntu, you can install ROS. The most current version of ROS is ROS hydro, but currently the Stingray's software is in ROS Groovy [are we switching? decide on this -- which to install!].

3. Go through ROS Tutorials

ROS supports C++, Python, and Lisp. We develop our software primarily in C++ and Python. Once you have ROS installed, work through the ROS tutorials in your language of choice to get familiar with ROS.
